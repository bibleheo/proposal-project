04 · WHAT'S NEW 2026

ZWCAD 2026, 무엇이 새로워졌나

PARAMETRIC
파라메트릭 구속조건
2D 객체·동적 블록에 기하/치수 구속조건을 추가해 다양한 크기 부품을 빠르게 생성
SMART DIM
지능형 스마트 치수
객체를 지능적으로 인식해 가장 적합한 치수 유형을 자동 생성 (DIM)
SMART MATCH
스마트 매칭 · 배치 블록
위치·크기·각도가 달라도 동일 객체를 일괄 선택·교체
SMART PLOT
스마트 플롯
도면 프레임 자동 검색, 축척에 맞는 배치를 일괄 생성
OUTPUT
PDF · DWF 출력 효율
파일 크기 50% 이상 압축, 새로운 출력 품질 옵션 추가
COMPATIBILITY
호환성 강화
PC5 외 PC3 플로터 형식 호환 등 출력 호환성 개선
