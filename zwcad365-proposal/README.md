# ZWCAD 365 제품 제안서 (BLUETNS)

ZWCAD 365 제품 소개·제안 랜딩페이지. 정적 사이트로, GitHub Pages에 그대로 호스팅할 수 있습니다.

## 폴더 구조
```
index.html      # 메인 페이지 (이 파일 하나가 진입점)
support.js      # 렌더링 런타임 (수정 불필요)
img/            # 로고·배경·구성도
pages/          # 기능 UI 캡처
```

## 로컬에서 보기
폴더 전체를 받은 뒤 `index.html`을 브라우저로 열면 됩니다.

## GitHub Pages 배포
1. 새 저장소를 만들고 이 폴더의 **모든 파일**을 업로드합니다.
2. 저장소 **Settings → Pages → Branch: `main` / `/ (root)` → Save**.
3. 1~2분 뒤 `https://<사용자명>.github.io/<저장소명>/` 에서 공개됩니다.

## 담당자 관리 / 단일 HTML 내보내기
- 우측 상단 **톱니(설정)** 아이콘 → 비밀번호 `bts7782!`
- 담당자 체크/수정 후 **단일 HTML로 내보내기** → 고객사 개별 발송용 파일 생성
- 우측 상단 **PDF** 버튼 → 가로 방향 PDF 저장
